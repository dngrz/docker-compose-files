# Informix 14.10.FC2DE Database Server

_Servidor de base de datos informix 12 para desarrollo de aplicaciones._ 

Basado en la imagen:

```
docker pull ibmcom/informix-developer-database:14.10.FC2DE
```

_Para construir la imagen se utilizó el siguiente comando._ 

```
docker build -t clodoaldosanchez/informix:14.10.FC2DE .
```
